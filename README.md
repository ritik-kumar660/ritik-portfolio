# 🚀 Ritik Kumar — Portfolio

Personal developer portfolio built with **React + Vite + Tailwind CSS**.

## 🔧 Tech Stack
- ⚡ **Vite** + **React 18**
- 🎨 **Tailwind CSS**
- 📧 **EmailJS** for contact form
- 🎞️ **react-fast-marquee** for skill scroll
- 🧩 **react-icons** + **lucide-react**
